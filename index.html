Sauvegarde & synchro

Les données sont enregistrées automatiquement dans ce navigateur. Le JSON sert de sauvegarde et de synchro PC ↔ mobile : exporte ici, importe là-bas. L'export détaille tout : pesées, mensurations, repas réellement mangés (avec quantités), séances (charge × reps par série), programmes et liste de courses. L'import fusionne intelligemment (par date / id, le fichier importé gagne).

Pour importer un programme seul, le fichier JSON peut ne contenir que la clé programmes : [{"id","nom","jours":[{"id","nom","exercices":[{"nom","series","reps","note"}]}]}].
